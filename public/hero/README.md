# Zdjęcia do slidera (strona główna)

## Desktop
- `slide-1.jpg` … `slide-4.jpg` – używane na ekranach > 768px.

## Mobile (opcjonalnie)
- `slide-1-mobile.jpg` … `slide-4-mobile.jpg` – używane na ekranach ≤ 768px.

Jeśli brak pliku `-mobile`, na mobile wyświetlane jest to samo zdjęcie co na desktop (`slide-X.jpg`).

Wersje mobile warto przyciąć do proporcji pionowych (np. 828×1080 px), żeby lepiej wypełniały ekran telefonu.
